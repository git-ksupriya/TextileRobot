Project Overview

The Textile Designing Robot is a robotic arm designed to automate simple textile painting and stamping tasks. The system detects the colour of the cloth using a colour sensor and then performs predefined painting patterns using a sponge end-effector. The goal of the project is to demonstrate how robotic manipulation, sensor integration, and basic kinematic principles can be applied to automate repetitive textile decoration tasks.

Project Objectives

  • Design a simple 4-DOF robotic arm for textile painting.
  • Automate basic stamping and painting operations.
  • Use servo motors for precise motion control.
  • Detect cloth colour and select paint accordingly.
  • Demonstrate robotic control using predefined motion sequences.
  • Reduce manual effort in textile stamping.

Robot Specifications

  • Type: 3R planar manipulator with rotating base
  • Degrees of Freedom: 4 DOF
  • Workspace: Fixed predefined workspace
  • End Effector: Sponge-based painting tool
  • Controller: Raspberry Pi Pico

Hardware Components Used

  • 4 Servo Motors
  • Raspberry Pi Pico Microcontroller
  • TCS3200 / TCS230 Color Sensor
  • Sponge End Effector
  • Robot Arm Frame and Links
  • 5V 2A Power Supply

Software and Programming

  • Programming Language: MicroPython
  • IDE: Thonny IDE
  • Servo Control: PWM Signals
  • Sensor Processing: RGB frequency analysis

Simulation Environment

Before implementing the system on hardware, the robotic arm behaviour and motion planning were tested in simulation using CoppeliaSim. The simulation allowed us to verify the robot workspace, joint movements, and painting trajectories before deploying them to the physical robot.

System Pipeline

1.System Initialization​
The Raspberry Pi Pico initializes the servo motors of the robotic arm, the TCS230 color sensor, and predefined painting patterns stored in memory.​
2.Cloth Placement and Detection​
The fabric is placed in the fixed workspace, ensuring proper alignment beneath the color sensor for accurate color detection.​
3.Color Sensing using TCS230/TCS3200​
The color sensor emits light onto the cloth surface and measures the reflected intensity for red, green, and blue components by switching internal photodiode filters.​
4.Signal Processing and Color Identification​
The sensor outputs frequency signals corresponding to RGB values, which are processed by the microcontroller to identify the dominant cloth color.​
5.Paint Selection Logic​
Based on the detected color, the controller selects the appropriate paint or painting pattern from predefined mappings.​
6.Motion Planning of Robotic Arm​
Predefined joint angles and trajectories for the robotic arm are retrieved, ensuring smooth and collision-free movement within the fixed workspace.​
7.Initial Paint Dipping Operation​
Before painting begins, the arm moves the sponge end effector to a specified paint container location and dips it into the selected paint.​
8.Painting Operation​
The robotic arm performs the painting pattern with controlled servo movements while maintaining continuous contact between the sponge and the cloth.​
9.Periodic Paint Re-Dipping (at Regular Intervals)​
At predefined time or step intervals, the arm temporarily pauses the painting motion and moves the sponge back to the specified paint location to re-dip it, ensuring sufficient paint is available and preventing paint depletion.​
10.Resume Painting Pattern​
After re-dipping, the arm returns to the last painting position and continues executing the remaining pattern smoothly.​
11.Pattern Completion and Reset​
Once the pattern is completed, the arm returns to its home position.​

Concepts Used

  • Robot Kinematics
  • Inverse Kinematics
  • Servo Motor Control using PWM
  • Sensor Integration
  • Colour Detection using RGB Frequency Output
  • Predefined Motion Planning
