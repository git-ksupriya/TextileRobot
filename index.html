Project Overview

The Textile Designing Robot is a robotic arm designed to automate simple textile painting and stamping tasks. The system detects the colour of the cloth using a colour sensor and then performs predefined painting patterns using a sponge end-effector. The goal of the project is to demonstrate how robotic manipulation, sensor integration, and basic kinematic principles can be applied to automate repetitive textile decoration tasks.

Project Objectives

  • Design a simple 4-DOF robotic arm for textile painting.
  • Automate basic stamping and painting operations.
  • Use servo motors for precise motion control.
  • Detect cloth colour and select paint accordingly.
  • Demonstrate robotic control using predefined motion sequences.
  • Reduce manual effort in textile stamping.

Robot Specifications

  • Type: 3R planar manipulator with rotating base
  • Degrees of Freedom: 4 DOF
  • Workspace: Fixed predefined workspace
  • End Effector: Sponge-based painting tool
  • Controller: Raspberry Pi Pico

Hardware Components Used

  • 4 Servo Motors
  • Raspberry Pi Pico Microcontroller
  • TCS3200 / TCS230 Color Sensor
  • Sponge End Effector
  • Robot Arm Frame and Links
  • 5V 2A Power Supply

Software and Programming

  • Programming Language: MicroPython
  • IDE: Thonny IDE
  • Servo Control: PWM Signals
  • Sensor Processing: RGB frequency analysis

Simulation Environment

Before implementing the system on hardware, the robotic arm behaviour and motion planning were tested in simulation using CoppeliaSim. The simulation allowed us to verify the robot workspace, joint movements, and painting trajectories before deploying them to the physical robot.

System Pipeline

  1. System initializes servos and colour sensor.
  2. Cloth is placed in the robot workspace.
  3. The colour sensor detects cloth colour.
  4. The controller processes RGB values.
  5. A suitable paint colour or pattern is selected.
  6. The robotic arm dips the sponge into paint.
  7. The robot executes the predefined painting pattern.
  8. The sponge periodically re-dips in paint.
  9. Pattern completes and robot returns to home position.

Concepts Used

  • Robot Kinematics
  • Inverse Kinematics
  • Servo Motor Control using PWM
  • Sensor Integration
  • Colour Detection using RGB Frequency Output
  • Predefined Motion Planning
