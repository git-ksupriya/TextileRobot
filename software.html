Robot Software

The real robot is programmed using MicroPython and executed on a Raspberry Pi Pico using the Thonny IDE. The program performs two main tasks: colour detection using a colour sensor and robotic arm motion control using inverse kinematics.

Colour Detection Algorithm

The colour sensor outputs a frequency proportional to the intensity of red, green and blue light. The program measures these frequencies and determines the dominant colour. 


function detect_color()

    set sensor filter to RED
    wait for stabilization
    measure red frequency

    set sensor filter to GREEN
    wait for stabilization
    measure green frequency

    set sensor filter to BLUE
    wait for stabilization
    measure blue frequency

    if red frequency is smallest
        return RED

    else if green frequency is smallest
        return GREEN

    else if blue frequency is smallest
        return BLUE

    else
        return UNKNOWN

end function

Inverse Kinematics Algorithm

To move the robotic arm to a desired position in space, inverse kinematics is used to compute the joint angles required to reach that position. 


function inverse_kinematics(x,y,z)

    compute base angle
    base = atan2(y,x)

    compute planar distance
    r = sqrt(x² + y²)

    compute elbow angle using cosine rule

    compute shoulder angle using triangle geometry

    convert angles from radians to degrees

    clamp angles within servo limits (0 to 180)

    return base, shoulder, elbow

end function

Trajectory Execution


start position = (x0,y0,z0)
end position = (xf,yf,zf)

for i from 0 to steps

    t = i / steps

    x = x0 + t*(xf-x0)
    y = y0 + t*(yf-y0)
    z = z0 + t*(zf-z0)

    angles = inverse_kinematics(x,y,z)

    move servos to angles

end for

Running the Program in Thonny

  1. Connect Raspberry Pi Pico to the computer using USB.
  2. Open the Thonny IDE.
  3. Select MicroPython (Raspberry Pi Pico) as the interpreter.
  4. Copy the Python program into the editor.
  5. Save the program to the Pico as main.py.
  6. Click Run to execute the program.
  7. The robot will begin colour detection and motion execution.
