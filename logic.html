Mathematical Logic

This section describes the mathematical modelling used for controlling the robotic arm and detecting colours during the textile painting process.

Robot Configuration

The proposed painting robot is modelled as a 3R planar manipulator mounted on a rotational base.

The system consists of the following joints:

  • Joint 0 (Base Joint) – Revolute joint allowing the entire arm to rotate about the Z-axis.
  • Joint 1, Joint 2, Joint 3 – Three revolute joints forming a planar R-R-R manipulator.
  • Links – Three rigid links with lengths L₁, L₂ and L₃.

Therefore the robot has 4 Degrees of Freedom (DOF).

The base of the robot is assumed to be located at the origin:

O = (0 , 0 , 0)

The desired target point on the cloth surface is defined as:

P = (x , y , z)

Base Rotation

The base joint rotates the entire robotic arm toward the target point in the XY plane.

The base rotation angle is calculated as:

θ₀ = atan2(y , x)

This aligns the planar arm with the target direction.

Projection into Planar Workspace

After base rotation, the target point is projected into the working plane of the robotic arm.

The horizontal distance from the base is:

r = √(x² + y²)

The planar arm must therefore reach the point:

(r , z)

Forward Kinematics

Let θ₁, θ₂ and θ₃ be the joint angles of the planar arm and L₁, L₂, L₃ be the link lengths.

The end-effector position in the plane is given by:

xp = L₁ cos(θ₁) + L₂ cos(θ₁ + θ₂) + L₃ cos(θ₁ + θ₂ + θ₃)

zp = L₁ sin(θ₁) + L₂ sin(θ₁ + θ₂) + L₃ sin(θ₁ + θ₂ + θ₃)

For a valid solution:

xp = r

zp = z

Inverse Kinematics

The inverse kinematics problem consists of finding the joint angles:

θ₀ , θ₁ , θ₂ , θ₃

given the desired target point (x , y , z) and the link lengths L₁ , L₂ , L₃.

The base angle θ₀ is computed first, followed by the planar arm angles using trigonometric relations.

Servo Angle Limits

Servo motors used in the robot can rotate only within a limited range.

Therefore all computed angles are clamped within:

0° ≤ θ ≤ 180°

before sending commands to the servo motors.

Linear Trajectory Planning

The robot moves smoothly between two points using linear interpolation.

x(t) = x₀ + t(xf − x₀)

y(t) = y₀ + t(yf − y₀)

z(t) = z₀ + t(zf − z₀)

where t varies from 0 to 1.

Servo PWM Control

Servo motors are controlled using PWM signals at approximately 50 Hz frequency.

The duty cycle determines the servo angle:

Duty = min_duty + (angle / 180) × (max_duty − min_duty)

Colour Detection Principle

The colour sensor contains photodiodes with red, green and blue filters. Each filter detects light intensity corresponding to its colour component.

The sensor outputs frequency values proportional to the detected light intensity.

By comparing the frequency values of the red, green and blue channels, the dominant colour of the object can be determined.

Calibration Considerations

  • Servo angles are clamped between safe operating limits.
  • Sensor readings require small delays for stabilization.
  • Linear interpolation ensures smooth robotic movement.
